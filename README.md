# 波シミュレータ（高校物理基礎）

高校物理基礎「波」分野を、生徒が自分のiPadで操作しながら視覚的に理解するためのシミュレーションツールです。

インストール不要・ログイン不要。URLを開くだけで動きます。各ファイルは外部リソースに依存しない単一のHTMLなので、一度読み込めばオフラインでも動作します。

## ツール

### ① 波の基本量シミュレータ — `index.html`

波形のグラフ（y–x）と、1点の振動のグラフ（y–t）を並べて見比べるツールです。

- 振幅 A・波長 λ・振動数 f をスライダーで変更（v = fλ と 周期 T = 1/f を自動計算して表示）
- 波の上をタップすると注目する媒質を選べます。選ぶと自動で一時停止し、再生すると y–t グラフが t = 0 から描き出されます
- v・T の表示はチェックボックスで隠せます（生徒に関係を発見させたいとき用）
- t = 0 の波形を点線で残せます

### ② 波の基本式を見つけよう — `discover.html`

①から y–t グラフと v・T の表示を取り除いた、探究用の機能制限版です。生徒自身に v = fλ を見つけさせることを目的としています。

- 進んだ距離（位置の目盛り）と経過時間から、生徒が自分で v を求められます
- 波を「線」で表示するか「媒質（点）の集まり」で表示するかを切り替えられます。点は x を固定して y だけが振動するので、媒質が進んでいないことが分かります

### ③ 縦波の横波表示 変換ツール — `longitudinal.html`

縦波（疎密波）の実際の粒子の動きと、教科書でおなじみの「横波表示」への変換を並べて見比べるツールです。

- 上段：媒質粒子の列が実際に左右に振動し、疎密がそのまま見えます
- 下段：粒子のx方向の変位を、位置に対して縦軸にプロットした横波表示グラフ（上段と完全に同期）
- 疎密の色分け表示（赤 = 密、青 = 疎）を切り替え可能
- 粒子をタップすると注目点を選べます。選んだ点の変位と、今どちら向きに動いているか（→/←）を表示します
- 振幅は波長に応じて自動的に上限が制限されます（粒子同士が理論上すり抜けないようにするため）
- 「変換のしくみ」モード：波を止めて、変換の手順を段階的に見せます。①初期状態（等間隔の媒質）→ ②変位を横向きの矢印で表す → ③左の粒子から1個ずつ矢印を90°回転（右向きの変位＝赤は上へ、左向き＝青は下へ。回転の軌跡を90°の円弧で表示）→ ④先端をつないで横波表示。「1個すすめる」で発問しながら手動で進めることもできます

### ④ 弦の振動・気柱の共鳴 — `resonance.html`

弦の固有振動と気柱の共鳴を、共通の「共鳴度」モデルで探究できるツールです。

- **弦モード**：弦の長さ（3m/2m/1m）を選び、波長スライダーで倍振動を探します。共鳴すると弦が赤くなり、振動の名前（基本振動・n倍振動）を表示。高い倍振動でも操作しやすいよう、スライダーは波長ではなく倍振動の番号を基準に等間隔化。「前の共鳴／次の共鳴」ボタンで隣の倍振動へ正確にジャンプできます
- **気柱モード**：閉管（1m/3m/5m）・開管（1m/2m/3m）を切り替え、振動数スライダーで共鳴を探します。媒質を変位グラフで見るか、実際に前後に振動する粒子（縦波）として見るかをチェックボックスで切替可能
- 両モードとも、スライダーを共鳴度75%以上の位置で離すと自動的にぴったりの値へなめらかに収束します
- 長さ・波長・振動数・音速を大きな数値パネルで表示。図の中にも「←2.0m(2λ)→」のような寸法線を表示し、長さと波長の関係を直接見比べられます
- 共鳴度に応じて音量が変化する音（既定ミュート）付き

## 単位系

v = fλ が単位でも成り立つように統一しています（Hz × m = m/s）。

| 量 | 単位 |
|---|---|
| 変位 y・振幅 A・位置 x・波長 λ | m |
| 振動数 f | Hz |
| 速さ v | m/s |
| 時間 t・周期 T | s |

数値は概念理解のための目安のスケールです。

## 動作環境

iPad / iPhone / PC のモダンブラウザ。横向き（ランドスケープ）でスクロールなしに全体が収まるよう設計しています。

## ローカルで開く

ファイルをブラウザで直接開くか、フォルダ内で簡易サーバーを起動します。

```
python -m http.server 8000
```
